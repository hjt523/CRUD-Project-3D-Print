About

This App is part of my QA Academy Training course, the purpose of the task is to implement the skills learnt so far in a CRUD application ( Create,Read,Update,Delete) using Flask. I chose to build a 3D print status tracker both to support my personal hobby, and because a 3D print project can be decomposed into several non unique tasks that often get repeated across projects, ideal for a many-to-many database relationship required of the project.
